# biblio
Manage my bibliography.

## Quickstart

Everything uses Docker:

```
$ time ./build 
... processing scale-out.tex
... processing scale-up.tex
... processing security.tex
... processing storage.tex

real	0m11.158s
user	0m0.126s
sys	0m0.167s
$ ls *.pdf
scale-out.pdf	scale-up.pdf	security.pdf	storage.pdf
```
